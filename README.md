## Project Overview
This project is a basic fullstack skeleton structure containing separate backend and frontend folders. 

The backend is built using Node.js and Express and runs on Port 3000.  
The frontend folder is a placeholder for future labs.

## How to Run Backend

cd backend  
npm install  
npm start  

Server runs at:
http://localhost:3000

- `/` → Displays success message  
- `/about` → Displays student details  
- `/health` → Returns JSON status
